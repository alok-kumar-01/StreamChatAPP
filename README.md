# 🚀 MERN Video Calling & Chat Application

A modern full-stack real-time communication platform built using the **MERN Stack**. This application enables users to communicate through instant messaging, high-quality video calls, and collaborative features while ensuring security, scalability, and a seamless user experience.

---

## 📌 Overview

The **MERN Video Calling & Chat Application** is designed to provide a secure and responsive communication platform. It supports one-to-one and group conversations, real-time messaging, video conferencing, screen sharing, and call recording. The application follows modern development practices with scalable architecture and efficient state management.

---

## ✨ Features

- 💬 **Real-Time Messaging**
  - Instant messaging
  - Typing indicators
  - Emoji reactions

- 📹 **Video Calling**
  - One-to-one video calls
  - Group video calls

- 🖥️ **Screen Sharing & Recording**
  - Share your screen during meetings
  - Record video calls

- 🔐 **Authentication & Security**
  - JWT Authentication
  - Protected Routes
  - Secure API Access

- 🌍 **Language Exchange Platform**
  - Connect with users globally
  - 32 Unique UI Themes

- ⚡ **Responsive User Interface**
  - Built with React.js
  - Responsive design using Tailwind CSS

- 📦 **State Management**
  - Global state management using Zustand

- 🔄 **Efficient Data Fetching**
  - TanStack Query for API caching and synchronization

- 🛡️ **Robust Error Handling**
  - Frontend validation
  - Backend error handling

- ☁️ **Deployment Ready**
  - Optimized for cloud deployment

- 🚀 **Scalable Architecture**
  - Built with Express.js
  - MongoDB Database
  - Stream Video & Chat APIs

## 🚀 Getting Started

### Prerequisites

- Node.js
- MongoDB
- npm or yarn

---

### Installation

#### Clone the repository

```bash
git clone https://github.com/your-username/mern-video-chat-app.git
```

```bash
cd mern-video-chat-app
```

#### Install Backend Dependencies

```bash
cd backend
npm install
```

#### Install Frontend Dependencies

```bash
cd ../frontend
npm install
```

---

## ⚙️ Environment Variables

Create a `.env` file inside the backend folder.

```env
PORT=5000

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_jwt_secret

STREAM_API_KEY=your_stream_api_key

STREAM_API_SECRET=your_stream_api_secret

NODE_ENV=development
```

---

## ▶️ Run the Project

### Start Backend

```bash
cd backend
npm run dev
```

### Start Frontend

```bash
cd frontend
npm run dev
```
